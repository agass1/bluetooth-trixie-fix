# Скрипт исправления проблем с Bluetooth на Linux

**Причина создания:** Частые проблемы с Bluetooth после установки Debian Trixie с KDE Plasma.

Этот скрипт полностью очищает установленные пакеты и данные Bluetooth, а затем переустанавливает их. Он полезен, если у вас есть проблемы с подключением Bluetooth-устройств особенно в случаях когда не работает стек AD2P

## Как использовать

1. Скачайте скрипт `bluetooth_fix.sh`.
2. Сделайте скрипт исполняемым:

```bash
chmod +x bluetooth_fix.sh
```

3. Запустите скрипт с правами администратора:

```bash
sudo ./bluetooth_fix.sh
```

4. После завершения работы скрипт **спросит**, хотите ли вы перезагрузить систему. Для перезагрузки нужно ввести:

```
yes
```

Если ввести что-то другое — перезагрузка не произойдёт.

## Что делает скрипт

- Полностью удаляет все пакеты, связанные с Bluetooth.
- Удаляет старые данные конфигурации (`/var/lib/blueman` и `/var/lib/bluetooth`).
- Переустанавливает пакеты: `blueman`, `bluez`, `pulseaudio-module-bluetooth` вместе с зависимостями.
- Запрашивает у пользователя подтверждение на перезагрузку системы.
- Подготавливает систему к чистой работе с Bluetooth.
